# My-Python-Learning-Journey
## 项目简介
这是一个用于记录我Python学习历程的个人仓库。在这里，我存放了课程练习、个人项目以及学习笔记，旨在跟踪我的编程技能成长。本仓库目前包含两个我最满意的Python项目：一个实用的**学生成绩管理系统**和一个有趣的**数字猜谜游戏**。


## 项目一：学生成绩管理系统

### 项目描述
这是一个简单的学生成绩管理系统，可以记录学生的基本信息、各科成绩，并自动计算总分、平均分和等级评定。

### 功能特点
- 学生信息录入（姓名、语文、数学、英语成绩）
- 自动计算总分和平均分
- 智能等级评定（A/B/C/D/F五级制）
- 成绩统计表格输出
- 等级分布统计

按照提示依次输入2名学生的信息即可。

### 运行示例
请输入第1名学生的信息:
姓名: 张三
语文成绩: 85
数学成绩: 92
英语成绩: 88
请输入第2名学生的信息:
姓名: 李四
语文成绩: 76
数学成绩: 81
英语成绩: 79


## 项目二：数字猜谜游戏

### 项目描述
一个有趣的数字猜谜游戏，程序随机生成1-100的数字，玩家有7次机会猜中目标数字。

### 功能特点
- 随机数字生成（1-100范围）
- 7次尝试机会限制
- 智能提示（太大/太小）
- 输入验证和异常处理
- 胜负判定和结果展示
- 
### 游戏规则
1.  程序随机生成一个1-100的整数
2.  玩家有7次猜测机会
3.  每次猜测后程序会提示"太大"或"太小"
4.  在7次内猜中即获胜，否则失败

### 运行示例
欢迎来到数字猜谜游戏!
我已经想了一个1到100之间的数字，你有7次机会来猜中它。
第1次尝试，你还剩6次机会。
请输入你猜的数字: 50
你猜的数字太小了，再试试!
第2次尝试，你还剩5次机会。
请输入你猜的数字: 75
你猜的数字太大了，再试试!
